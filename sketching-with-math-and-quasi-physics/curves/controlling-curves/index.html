Controlling curves жӣІз·ҡгҒ®гӮігғігғҲгғӯгғјгғ«

We have explored different ways of defining and drawing curves, but we havenвҖҷt discussed the quality of curves much. Curves can have various visual characteristics. For example, some curves might look more organic and natural, while others might appear more geometric and mechanical. How can we create these different looks of curves?

гҒ“гҒ“гҒҫгҒ§жӣІз·ҡгӮ’е®ҡзҫ©гҒ—жҸҸз”»гҒҷгӮӢж§ҳгҖ…гҒӘж–№жі•гӮ’иҰӢгҒҰгҒҚгҒҫгҒ—гҒҹгҒҢгҖҒжӣІз·ҡгҒ®иіӘгҒ«гҒӨгҒ„гҒҰгҒҜгҒӮгҒҫгӮҠиӯ°и«–гҒ—гҒҰгҒҚгҒҫгҒӣгӮ“гҒ§гҒ—гҒҹгҖӮжӣІз·ҡгҒ«гҒҜеӨҡж§ҳгҒӘиҰӢгҒҹзӣ®гҒ®зү№еҫҙгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮдҫӢгҒҲгҒ°жӣІз·ҡгҒ«гҒҜжңүж©ҹзҡ„гҒ§иҮӘз„¶гҒ«иҰӢгҒҲгӮӢгӮӮгҒ®гӮӮгҖҒгӮҲгӮҠе№ҫдҪ•еӯҰзҡ„гҒ§ж©ҹжў°зҡ„гҒ«иҰӢгҒҲгӮӢгӮӮгҒ®гӮӮгҒӮгӮҠгҒҫгҒҷгҖӮгҒ“гӮҢгӮүгҒ®з•°гҒӘгӮӢжӣІз·ҡгҒ®иҰӢгҒҹзӣ®гҒҜгҒ©гҒҶгҒ—гҒҹгӮүдҪңгӮҢгӮӢгҒ®гҒ§гҒ—гӮҮгҒҶгҒӢгҖӮ

Cubic class

Cubic гӮҜгғ©гӮ№

For most of the examples on this page, we will use the same Cubic class that represents a Cubic BГ©zier curve we discussed previously as a way to define curves as a baseline. For sketching purposes, where precision is not too important, you can combine multiple cubic BГ©zier curves to approximate any curve.

гҒ“гҒ®гғҡгғјгӮёгҒ®гҒ»гҒЁгӮ“гҒ©гҒ®дҫӢгҒ§гҒҜгҖҒгҒҷгҒ§гҒ«и§ҰгӮҢгҒҹгҖҒ3ж¬ЎгғҷгӮёгӮ§жӣІз·ҡгӮ’иЎЁгҒҷ Cubic гӮҜгғ©гӮ№гӮ’жӣІз·ҡгӮ’е®ҡзҫ©гҒҷгӮӢж–№жі•гҒ®еҹәзӨҺгҒЁгҒ—гҒҰдҪҝгҒ„гҒҫгҒҷгҖӮгӮ№гӮұгғғгғҒзӣ®зҡ„гҒ§зІҫеәҰгҒҢгҒқгӮҢгҒ»гҒ©йҮҚиҰҒгҒ§гҒӘгҒ„е ҙйқўгҒ§гҒҜгҖҒиӨҮж•°гҒ®3ж¬ЎгғҷгӮёгӮ§жӣІз·ҡгӮ’зө„гҒҝеҗҲгӮҸгҒӣгӮҢгҒ°д»»ж„ҸгҒ®жӣІз·ҡгӮ’иҝ‘дјјгҒ§гҒҚгҒҫгҒҷгҖӮ

class Cubic {
  constructor(a0, c0, c1, a1) {
      this.a0 = a0.copy();
      this.c0 = c0.copy();
      this.c1 = c1.copy();
      this.a1 = a1.copy();
  }

  getPointAt(t) {
      const u = 1 - t;
      return new p5.Vector(
          this.a0.x * (u * u * u) + this.c0.x * (3 * t * u * u) +
          this.c1.x * (3 * t * t * u) + this.a1.x * (t * t * t),
          this.a0.y * (u * u * u) + this.c0.y * (3 * t * u * u) +
          this.c1.y * (3 * t * t * u) + this.a1.y * (t * t * t)
      );
  }

  draw() {
    const resolution = 128;
    strokeCap(ROUND);
    beginShape();
    for (let i = 0; i <= resolution; i++) {
      const t = i / resolution;
      const p = this.getPointAt(t);
      vertex(p.x, p.y);
    }
    endShape();
  }

Natural Spline

гғҠгғҒгғҘгғ©гғ«гӮ№гғ—гғ©гӮӨгғі

BГ©zier curves give you detailed control using control points, but creating a complex shape requires many of them, which can be very troublesome to configure. Additionally, combining multiple BГ©zier curves does not always result in smooth connections. Natural spline is a very handy method when you just want a smooth curve nicely goes through a bunch of points.

гғҷгӮёгӮ§жӣІз·ҡгҒҜеҲ¶еҫЎзӮ№гӮ’з”ЁгҒ„гҒҰзҙ°гҒӢгҒӘеҲ¶еҫЎгҒҢгҒ§гҒҚгҒҫгҒҷгҒҢгҖҒиӨҮйӣ‘гҒӘеҪўгӮ’дҪңгӮӢгҒ«гҒҜеӨҡгҒҸгҒ®еҲ¶еҫЎзӮ№гҒҢеҝ…иҰҒгҒ§гҖҒиЁӯе®ҡгҒҢгҒЁгҒҰгӮӮйқўеҖ’гҒ§гҒҷгҖӮгҒҫгҒҹгҖҒиӨҮж•°гҒ®гғҷгӮёгӮЁжӣІз·ҡгӮ’зө„гҒҝеҗҲгӮҸгҒӣгҒҰгӮӮгҖҒеҝ…гҒҡгҒ—гӮӮж»‘гӮүгҒӢгҒ«з№ӢгҒҢгҒЈгҒҰгҒҸгӮҢгӮӢгӮҸгҒ‘гҒ§гҒҜгҒӮгӮҠгҒҫгҒӣгӮ“гҖӮгғҠгғҒгғҘгғ©гғ«гӮ№гғ—гғ©гӮӨгғігҒҜгҖҒгҒҹгҒҸгҒ•гӮ“гҒ®зӮ№гӮ’ж»‘гӮүгҒӢгҒ«гҒҶгҒҫгҒҸйҖҡйҒҺгҒҷгӮӢжӣІз·ҡгҒҢж¬ІгҒ—гҒ„гҒЁгҒҚгҒ«йқһеёёгҒ«дҫҝеҲ©гҒӘж–№жі•гҒ§гҒҷгҖӮ

Continuity of Curves

жӣІз·ҡгҒ®йҖЈз¶ҡжҖ§

To understand what the natural spline is, letвҖҷs briefly discuss what being smooth means. To measure how smooth a curve is in a mathematical sense, you can take the derivative of the function that represents the curve. The first derivative shows the rate of change of position, and the second derivative shows the rate of change of the rate of change.

гғҠгғҒгғҘгғ©гғ«гӮ№гғ—гғ©гӮӨгғігҒҢдҪ•гҒӘгҒ®гҒӢгӮ’зҗҶи§ЈгҒҷгӮӢгҒҹгӮҒгҒ«гҖҒгҖҢж»‘гӮүгҒӢгҒ§гҒӮгӮӢгҖҚгҒЁгҒ„гҒҶгҒ“гҒЁгҒ®ж„Ҹе‘ігҒ«гҒӨгҒ„гҒҰз°ЎеҚҳгҒ«иӯ°и«–гҒ—гҒҫгҒ—гӮҮгҒҶгҖӮжӣІз·ҡгҒ®ж»‘гӮүгҒӢгҒ•гӮ’ж•°еӯҰзҡ„гҒ«жё¬гӮӢгҒ«гҒҜгҖҒгҒқгҒ®жӣІз·ҡгӮ’иЎЁгҒҷй–ўж•°гҒ®еҫ®еҲҶгӮ’еҸ–гӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгҒҫгҒҷгҖӮжңҖеҲқгҒ®еҫ®еҲҶгҒҜдҪҚзҪ®гҒ®еӨүеҢ–зҺҮгӮ’зӨәгҒ—гҖҒ2еәҰзӣ®гҒ®еҫ®еҲҶгҒҜгҒқгҒ®еӨүеҢ–зҺҮгҒ®еӨүеҢ–гӮ’зӨәгҒ—гҒҫгҒҷгҖӮгҒ“гҒ®гӮҲгҒҶгҒ«гҒҷгӮӢгҒЁгҖҒз•°гҒӘгӮӢгғ¬гғҷгғ«гҒ®йҖЈз¶ҡжҖ§гӮ’иҖғгҒҲгӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгҒҫгҒҷгҖӮ

  • C0C^0C0 Continuity: The curve itself is continuous. There are no breaks or gaps. You can imagine a line without lifting your pen. There is no gap in this line, but there can be sharp corners and turns that may not appear smooth.

  • C1C^1C1 Continuity: The first derivative of the curve is continuous. This means the curve has no sharp corners or cusps; it changes direction smoothly. You can think of a smoothly curving road without any sudden turns.

  • C2C^2C2 Continuity: The second derivative of the curve is continuous. This means that even the rate at which the curve changes direction is continuous. Roller coasters are usually designed to have C2C^2C2 continuity as much as possible to avoid too sudden a change in acceleration, which can be uncomfortable or even harmful to the passengers. If it is not C2C^2C2 continuous, sudden changes in G-forces might slam you into the seat or cause whiplash.

  • C0C^0C0йҖЈз¶ҡ: жӣІз·ҡгҒ«йҖ”еҲҮгӮҢгӮ„йҡҷй–“гҒҢгҒӘгҒҸйҖЈз¶ҡгҒ—гҒҰгҒ„гҒҫгҒҷгҖӮгғҡгғігӮ’жҢҒгҒЎдёҠгҒ’гҒҡгҒ«жҸҸгҒ„гҒҹз·ҡгӮ’жғіеғҸгҒ—гҒҰгҒҸгҒ гҒ•гҒ„гҖӮгҒ“гҒ®з·ҡгҒ«гҒҜйҖ”еҲҮгӮҢгҒҹз®ҮжүҖгҒҜгҒӮгӮҠгҒҫгҒӣгӮ“гҒҢгҖҒж»‘гӮүгҒӢгҒ«иҰӢгҒҲгҒӘгҒ„йӢӯгҒ„и§’гӮ„жӣІгҒҢгҒЈгҒҹйғЁеҲҶгҒҢгҒӮгӮӢгҒӢгӮӮгҒ—гӮҢгҒҫгҒӣгӮ“гҖӮ

  • C1C^1C1 йҖЈз¶ҡ: жӣІз·ҡгҒ®1ж¬Ўеҫ®еҲҶгҒҢйҖЈз¶ҡгҒ§гҒҷгҖӮгҒ“гӮҢгҒҜгҖҒжӣІз·ҡгҒҢж»‘гӮүгҒӢгҒ«ж–№еҗ‘гӮ’еӨүгҒҲйӢӯгҒ„и§’гӮ„е°–гҒЈгҒҹзӮ№гҒҢгҒӘгҒ„гҒ“гҒЁгӮ’ж„Ҹе‘ігҒ—гҒҫгҒҷгҖӮзӘҒз„¶гҒ®жҠҳгӮҢжӣІгӮҠгҒҢгҒӘгҒ„ж»‘гӮүгҒӢгҒӘйҒ“и·ҜгӮ’жғіеғҸгҒ—гҒҰгҒҝгҒҫгҒ—гӮҮгҒҶгҖӮ

  • C2C^2C2 йҖЈз¶ҡ: жӣІз·ҡгҒ®2ж¬Ўеҫ®еҲҶгҒҢйҖЈз¶ҡгҒ§гҒҷгҖӮгҒ“гӮҢгҒҜгҖҒжӣІз·ҡгҒҢеҗ‘гҒҚгӮ’еӨүгҒҲгӮӢйҖҹгҒ•гҒ•гҒҲгӮӮж»‘гӮүгҒӢгҒ§гҒӮгӮӢгҒ“гҒЁгӮ’ж„Ҹе‘ігҒ—гҒҫгҒҷгҖӮгӮёгӮ§гғғгғҲгӮігғјгӮ№гӮҝгғјгҒҜгҖҒд№—е®ўгҒ«дёҚеҝ«гҒ•гӮ„еҚұйҷәгӮ’дёҺгҒҲгҒӘгҒ„гӮҲгҒҶгҖҒгҒ§гҒҚгӮӢгҒ гҒ‘C2C^2C2йҖЈз¶ҡжҖ§гӮ’жҢҒгҒӨгӮҲгҒҶгҒ«иЁӯиЁҲгҒ•гӮҢгҒҰгҒ„гҒҫгҒҷгҖӮC2C^2C2йҖЈз¶ҡгҒ§гҒӘгҒ„е ҙеҗҲгҖҒжҖҘгҒӘGгҒ®еӨүеҢ–гҒ§дҪ“гӮ’еёӯгҒ«жү“гҒЎгҒӨгҒ‘гҒҹгӮҠгҖҒгӮҖгҒЎжү“гҒЎз—ҮгҒ«гҒӘгҒЈгҒҹгӮҠгҒҷгӮӢгҒӢгӮӮгҒ—гӮҢгҒҫгҒӣгӮ“гҖӮ

A natural spline is a series of curves that pass through all the given points and is C2C^2C2 continuous.

Hobby Curve

Hobby жӣІз·ҡ

While a natural spline is mathematically smooth, it may not always appear to be the smoothest line to the human eye. There are various ways to think about the smoothness of a curve. One approach is to minimize the curvature at any given point, ensuring that each point on the curve looks closer to straight line as possible. The Hobby curve algorithm is designed for this. Using a simple calculation called вҖңmock curvature,вҖқ which is an approximation of the more computationally heavy вҖңtrue curvature,вҖқ this method can create a pleasing line that goes through a series of points.

гғҠгғҒгғҘгғ©гғ«гӮ№гғ—гғ©гӮӨгғігҒҜж•°еӯҰзҡ„гҒ«ж»‘гӮүгҒӢгҒ§гҒҷгҒҢгҖҒеҝ…гҒҡгҒ—гӮӮдәәгҒ«гҒЁгҒЈгҒҰжңҖгӮӮж»‘гӮүгҒӢгҒӘз·ҡгҒ«иҰӢгҒҲгӮӢгҒЁгҒҜйҷҗгӮҠгҒҫгҒӣгӮ“гҖӮжӣІз·ҡгҒ®ж»‘гӮүгҒӢгҒ•гҒ«гҒӨгҒ„гҒҰиҖғгҒҲгӮӢж–№жі•гҒҜиүІгҖ…гҒӮгӮҠгҒҫгҒҷгҖӮгҒқгҒ®1гҒӨгҒҜгҖҒд»»ж„ҸгҒ®зӮ№гҒ§гҒ®жӣІзҺҮгӮ’жңҖе°ҸйҷҗгҒ«жҠ‘гҒҲгҖҒжӣІз·ҡдёҠгҒ®гҒқгӮҢгҒһгӮҢгҒ®зӮ№гҒҢгҒ§гҒҚгӮӢгҒ гҒ‘зӣҙз·ҡгҒ«иҝ‘гҒҸиҰӢгҒҲгӮӢгӮҲгҒҶгҒ«гҒҷгӮӢгӮўгғ—гғӯгғјгғҒгҒ§гҖҒгҒ“гӮҢгҒҢHobbyжӣІз·ҡгҒ®гӮўгғ«гӮҙгғӘгӮәгғ гҒҢзӣ®жҢҮгҒҷж»‘гӮүгҒӢгҒ•гҒ§гҒҷгҖӮгҒ“гҒ®ж–№жі•гҒ§гҒҜгҖҒиЁҲз®—гҒҢйҮҚгҒ„зңҹгҒ®жӣІзҺҮгӮ’жұӮгӮҒгӮӢд»ЈгӮҸгӮҠгҒ«гҖҒгҒқгҒ®з°ЎеҚҳгҒӘиҝ‘дјјгҒ§гҒӮгӮӢз–‘дјјжӣІзҺҮгӮ’з”ЁгҒ„гҒҰгҖҒдёҺгҒҲгӮүгӮҢгҒҹзӮ№гӮ’йҖҡгӮӢиҰӢгҒҹзӣ®гҒ®иүҜгҒ„жӣІз·ҡгӮ’дҪңгӮҢгҒҫгҒҷгҖӮ

The Hobby curve feels softer and more human, somewhat like Henri MatisseвҖҷs stroke drawn with a very long brush.

Rounding Corners

и§’гӮ’дёёгӮҒгӮӢ

For a more geometric and uniform look, one of the common methods is to replace the joints of line segments with circular arcs. If you have used drawing tools like Adobe Illustrator, Sketch, or Figma, you probably know how you can round corners in these tools by specifying the radius.

гӮҲгӮҠе№ҫдҪ•еӯҰзҡ„гҒ§еқҮдёҖгҒӘиҰӢгҒҹзӣ®гӮ’дҪңгӮӢгҒҹгӮҒгҒ«гӮҲгҒҸз”ЁгҒ„гӮүгӮҢгҒҷж–№жі•гҒЁгҒ—гҒҰгҖҒз·ҡеҲҶгҒҢгҒӨгҒӘгҒҢгӮӢйғЁеҲҶгӮ’еҶҶеј§гҒ§зҪ®гҒҚжҸӣгҒҲгӮӢжүӢжі•гҒҢгҒӮгӮҠгҒҫгҒҷгҖӮAdobe IllustratorгҖҒSketchгҖҒFigmaгҒӘгҒ©гҒ®гғүгғӯгғјгӮӨгғігӮ°гғ„гғјгғ«гӮ’дҪҝгҒЈгҒҹгҒ“гҒЁгҒҢгҒӮгӮҢгҒ°гҖҒеҚҠеҫ„гӮ’жҢҮе®ҡгҒ—гҒҰи§’гӮ’дёёгӮҒгӮӢж©ҹиғҪгӮ’иҰӢгҒҹгҒ“гҒЁгҒҢгҒӮгӮӢгҒ§гҒ—гӮҮгҒҶгҖӮ

To find the position of the circle, you can draw a line through the corner and the center of the circle, and perpendicular lines from the center to the sides. Then you can find two congruent right triangles.

еҶҶгҒ®дҪҚзҪ®гӮ’жұӮгӮҒгӮӢгҒ«гҒҜгҖҒи§’гҒЁеҶҶгҒ®дёӯеҝғгӮ’йҖҡгӮӢзӣҙз·ҡгҒЁгҖҒдёӯеҝғгҒӢгӮүдёЎеҒҙгҒ®з·ҡгҒ«еһӮзӣҙгҒӘз·ҡгӮ’еј•гҒҸгҒЁгҖҒ2гҒӨгҒ®еҗҲеҗҢгҒӘзӣҙи§’дёүи§’еҪўгӮ’иҰӢгҒӨгҒ‘гӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгҒҫгҒҷгҖӮ

The distance from the corner to the points where the circle touches is:

и§’гҒӢгӮүеҶҶгҒЁдёЎеҒҙгҒ®з·ҡгҒ®жҺҘзӮ№гҒҫгҒ§гҒ®и·қйӣўгҒҜж¬ЎгҒ®гӮҲгҒҶгҒ«жұӮгӮҒгӮүгӮҢгҒҫгҒҷгҖӮ

a=rtanвҒЎ(Оё){\begin{aligned} a = \frac{r}{\tan(\theta)} \end{aligned}}a=tan(Оё)rвҖӢвҖӢ

The distance from the corner to the center is:

и§’гҒӢгӮүеҶҶгҒ®дёӯеҝғгҒҫгҒ§гҒ®и·қйӣўгҒҜж¬ЎгҒ®гӮҲгҒҶгҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ

b=rsinвҒЎ(Оё){\begin{aligned} b = \frac{r}{\sin(\theta)} \end{aligned}}b=sin(Оё)rвҖӢвҖӢ

To draw this in p5.js, we could use [arc()](https://p5js.org/reference/p5/arc/). But instead, the RoundedCorner class in the demo below takes three points and the radius and creates an array of Cubic objects. This way, we can treat all the different curves discussed on this page just as arrays of the same object, making it easy to connect or mix them.

гҒ“гӮҢгӮ’p5.jsгҒ§жҸҸгҒҸгҒ«гҒҜ[arc()](https://p5js.org/reference/p5/arc/)й–ўж•°гҒҢдҪҝгҒҲгҒҫгҒҷгҒҢгҖҒдёӢгҒ®гғҮгғўгҒ®RoundedCornerгӮҜгғ©гӮ№гҒ§гҒҜ3гҒӨгҒ®зӮ№гҒ®еҚҠеҫ„гӮ’гҒЁгҒЈгҒҰгҖҒCubicгҒ®й…ҚеҲ—гӮ’дҪңгӮҠгҒҫгҒҷгҖӮгҒ“гҒҶгҒҷгӮӢгҒ“гҒЁгҒ§гҖҒгҒ“гҒ®гғҡгғјгӮёгҒ§иҰӢгҒҰгҒҚгҒҹгҒҷгҒ№гҒҰгҒ®з•°гҒӘгӮӢжӣІз·ҡгӮ’еҗҢгҒҳгӮӘгғ–гӮёгӮ§гӮҜгғҲгҒ®й…ҚеҲ—гҒЁгҒ—гҒҰжүұгҒҶгҒ“гҒЁгҒҢгҒ§гҒҚгҖҒз№ӢгҒ’гҒҹгӮҠж··гҒңеҗҲгӮҸгҒӣгҒҹгӮҠгҒҷгӮӢгҒ“гҒЁгҒҢз°ЎеҚҳгҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ

To support this, the Cubic class has two new functions added. The circularArc() function creates an approximation of a circular arc as a cubic BГ©zier, and the straightLine() function let us treat a straight line segment in the same way. Note that a cubic BГ©zier cannot produce a circular arc precisely, but the approximation is good enough for most of our graphical needs.

гҒ“гӮҢгӮ’е®ҹзҸҫгҒҷгӮӢгҒҹгӮҒгҒ«гҖҒCubic гӮҜгғ©гӮ№гҒ«гҒҜ2гҒӨгҒ®ж–°гҒ—гҒ„й–ўж•°гӮ’иҝҪеҠ гҒ—гҒҫгҒ—гҒҹгҖӮcircularArc() й–ўж•°гҒҜеҶҶеј§гҒ®иҝ‘дјјгӮ’3ж¬ЎгғҷгӮёгӮ§жӣІз·ҡгҒЁгҒ—гҒҰдҪңжҲҗгҒ—гҖҒstraightLine() й–ўж•°гҒҜзӣҙз·ҡйғЁеҲҶзҡ„гӮ’еҗҢгҒҳж–№жі•гҒ§жүұгҒҲгӮӢгӮҲгҒҶгҒ«гҒ—гҒҫгҒҷгҖӮ3ж¬ЎгғҷгӮёгӮ§жӣІз·ҡгҒҜеҶҶеј§гӮ’жӯЈзўәгҒ«еҶҚзҸҫгҒ§гҒҚгҒҫгҒӣгӮ“гҒҢгҖҒеӨ§жҠөгҒ®гӮ°гғ©гғ•гғғгӮЈгӮҜзӣ®зҡ„гҒ«гҒҜеҚҒеҲҶгҒӘиҝ‘дјјгҒҢгҒ§гҒҚгҒҫгҒҷгҖӮ

The implementation is based Chet HaaseвҖҷs Medium post and the code for Android.

е®ҹиЈ…гҒҜChet HaaseгҒҢMediumгҒ«жҠ•зЁҝгҒ—гҒҹиЁҳдәӢгҒЁгҒЁAndroidгҒ®гӮігғјгғүгӮ’е…ғгҒ«гҒ—гҒҫгҒ—гҒҹгҖӮ

Applying this to more complex shapes, you can create various rounded shapes that still feel geometric and well-controlled.

гҒ“гӮҢгӮ’гӮҲгӮҠиӨҮйӣ‘гҒӘеҪўзҠ¶гҒ«йҒ©з”ЁгҒҷгӮҢгҒ°гҖҒдёёгҒҝгҒҢгҒӮгӮҠгҒӘгҒҢгӮүгҖҒе№ҫдҪ•еӯҰзҡ„гҒ§гӮҲгҒҸгӮігғігғҲгғӯгғјгғ«гҒ•гӮҢгҒҹйӣ°еӣІж°—гҒ®еҪўгӮ’ж§ҳгҖ…гҒ«дҪңгӮҢгҒҫгҒҷгҖӮ

Smoothening of the rounded corners

дёёгӮҒгҒҹи§’гӮ’ж»‘гӮүгҒӢгҒ«гҒҷгӮӢ

Rounded corners created in this way are not quite вҖңsmoothвҖқ in the sense that the curvature changes all of a sudden at the point the straight line meets the arc. If this were a road, you would need to steer the handle quite abruptly at the entrance and exit of the curve. To ease this, Apple introduced smoothing to the rounded corners in their design (To be fair, smoothed curves have been used widely in industrial design where NURBs are standard, but Apple has put emphasis on the soft curve in their product design and introduced it to UI language too). The AndroidвҖҷs shape library has an approximation of this too. Take a look at the code if you are interested.

But curves arenвҖҷt just about math. TheyвҖҷre also about aesthetics and functionality. Artists and designers have spent so much effort creating curves that are visually or tactually pleasing or express various ideas. Studying great works can teach us a lot about curves. For instance, you might be interested in how Raymond LoewyвҖҷs streamlined design reflected the fascination with progress and efficiency of the era, or you might want to explore how to emulate Ellsworth KellyвҖҷs soft rounded shapes with code. You could also explore topics like the design of roads and roller coasters, aerodynamics, or use of parametric approach in architecture and industrial design.
